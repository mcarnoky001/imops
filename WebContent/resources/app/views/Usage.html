Familiar: Can make things work by collaging code snippets together from Stack Overflow and other external referrences
Proficient: Occassional reference to Stack Overflow and external referrences, but have good understanding of what the code is doing
Excellent: Can code without Stack Overflow and minimal referrence to documentations
Expert: Understands internals of the language. For example, for Java, expert should understand internals of Java to the point they can read Java code and translate it to bytecode in his head
